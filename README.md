# Aria Demo — AI Receptionist for Med Spas
### By Cortana Solutions 🔷

A 60-second animated Instagram Reels demo video (9:16 ratio).

## Run on Replit
1. Click **Run** — server starts on port 3000
2. The preview opens automatically
3. Hit **▶ Restart** button to replay the animation

## Record the Video
1. Open the preview in a **new browser tab** (click the open-in-new-tab icon)
2. On Mac: Open **QuickTime** → File → New Screen Recording
3. Drag to select ONLY the 390×693 animation box
4. Record the full 60 seconds
5. Export as MP4 → import to CapCut → add music → post to Instagram Reels
